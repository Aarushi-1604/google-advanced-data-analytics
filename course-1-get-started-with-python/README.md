# Course 2: Getting Started with Python

## What this course focused on
This course built my foundation in Python for data analysis. I learned how to work with variables, data types, functions, conditionals, loops, and core data structures such as lists, dictionaries, and tuples. I also practiced writing clean, readable Python code that can be easily understood and reused by other data professionals.

Towards the end of the course, I was introduced to NumPy and pandas, two essential libraries used for handling and analyzing structured datasets in real-world data science projects.
